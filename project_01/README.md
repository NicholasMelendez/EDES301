# Project Name: EPIC Audio Analyzer

Welcome to the EPIC Audio Analyzer Github!

If you have not already wired up your pocketbeagle, 4 pin USB port, and your LCD display onto your breadboard, please see the [hackster.io](https://www.hackster.io/nm123/edes-301-epic-audio-analyzer-cc994b?auth_token=6399cccf68739bcc7e2da50530170805) page.


Before getting started please run these lines which allows the microphone to collect data:
> sudo apt-get update<
> sudo apt-get install libasound2-dev python3-pip<
> pip3 install pyaudio numpy<
