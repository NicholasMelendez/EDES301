# EDES301 Project_02: EtchASketch PCD Design




For my Second project in EDES301, I decided to design an Etch A Sketch PCB using KiCad. I was responsible for the KiCad Symbol and Footprint libraries, the KiCad Schematic, and the KiCad PCB layout. 

The EtchASketch was designed using the PocketBeagle Microprocessor, and it contains the following components:

* One Adafruit 2.8" TFT LCD with Touchscreen Breakout Board w/MicroSD Socket ( to Display EtchASketch Art)

* Two potentiometers ( To operate and draw using the EtchASketch)

* Three LEDs (Red, Green, and Blue for user color indication)

* Two Button (To swap colors and to reset the screen)

* One USB port (For future expansion)

* An Image of the GOAT Gojo Satoru



In addition to the KiCad files, the directory for project_02 also contains generated Gerber and Drill files for the PCB, A generated Bill of Materials document, Images of the Top and Bottom views of the PCB, a PDF of the Schematic, and a presentation containing the mechanical block diagram and the system block diagram
